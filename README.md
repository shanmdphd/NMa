# NMa: NONMEM Assistants

Acroedit을 사용하여 NONMEM을 더 쉽게 쓸 수 있다.

## 설명

- `nm74g64p.bat`: 배교수님이 만든 batch로서 각종 PDF를 만들어 낼 수 있다.
- `print4.exe`: 이 파일은 NONMEM output 파일에서 행의 시작이 0 혹은 1 으로 된 경우 이를 원래의 의도대로 변경해 준다.
- `AcroEditOpt`: 이 폴더에 있는 파일을 통해서 AcroEdit에서 syntax highlighting 기능을 사용할 수 있ㄷ.
- run UserTools.aut
- `docs`: 직접 제작한 매뉴얼이 들어있다.
